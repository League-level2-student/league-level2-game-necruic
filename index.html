1. Why I Made This Game

The game seemed easy to make but still had some complex elements.

2. What I Learned

na

3. Architecture of My Game

The game creates a 3 by 3 grid that allows 2 players to insert x and y symbols in them, with the game being won when one player connects 3 of their symbols either horizontally, diagonally, or vertically.

4. Technical Implementation

The game uses framedraw to create empty grids, then the paintComponent method creates rectangles to fill in the grids.

5. Most Challenging Part

The most challenging part of this project was when the game wouldn't switch from p1 to p2 and I fixed this by adding an end state and changing the currentstate from p1 to p2.

6. Future Fixes and Updates
